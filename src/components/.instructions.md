# Component Instructions

## Rules

- **Do NOT modify** files in `src/components/ui/` — these are shadcn/ui primitives.
- Use Phosphor Icons (`@phosphor-icons/react`), never Lucide or Heroicons.
- Use `cn()` from `@/lib/utils` to merge Tailwind classes.
- Use Framer Motion for animations; keep durations under 400ms.
- Use `sonner` for toast notifications.
- Keep components focused; extract reusable logic into `src/hooks/`.
- All Docker state mutations must go through the command parser — never mutate containers/images directly.

## Import Convention

```tsx
import { ComponentName } from '@/components/ui/component-name'
import { cn } from '@/lib/utils'
import { IconName } from '@phosphor-icons/react'
```
